# FinanceHub

FinanceHub é um organizador financeiro moderno construído com Angular. O foco do projeto é permitir que você registre receitas e despesas com facilidade, acompanhe o saldo disponível e visualize a distribuição financeira em um painel responsivo e agradável.

## Recursos

- Registro rápido de receitas e despesas.
- Visualização clara do saldo em tempo real.
- Histórico de movimentações com remoção rápida.
- Gráfico de pizza para entender a proporção entre ganhos e gastos.
- Navegação intuitiva entre Dashboard, Sobre e Contato.

## Desenvolvimento

Para iniciar o servidor local, execute:

```bash
ng serve
```

Abra `http://localhost:4200/` no navegador. A aplicação recarrega automaticamente ao salvar alterações.

## Estrutura

- `src/app/components/home` — dashboard principal de finanças.
- `src/app/components/about` — informações sobre o projeto.
- `src/app/components/contact` — contato e links.
- `src/app/components/sidebar` — navegação lateral.

## Build

Para compilar o projeto:

```bash
ng build
```

## Testes

Para rodar testes unitários:

```bash
ng test
```
